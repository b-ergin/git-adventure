Git Quiz

-What is Git and why is it so important?

Git is a system that helps developers work collaboratively.

-What’s a branch and why do we create them?

A version of the repository, which helps you work on a project without changing the main one. Once you are sure about the changes, you can merge them.

-What’s the working tree?

Directory you are working in.

What’s the staging area?

Staging area is where you control changes before committing them.

What’s the command to retrieve the latest version of your files from the remote serve?

git pull
